## 启动屏幕和应用图标

由于我们使用 Expo 创建 starter，更新应用图标和启动屏幕非常简单。您只需要更新 `assets` 文件夹中的应用图标和启动屏幕图像，然后运行 `expo prebuild` 命令来更新应用图标和启动屏幕。

由于我们为开发、staging 和生产环境支持多个变体，您需要 3 个不同的图标，但正确的解决方案是对每个环境使用相同的图标并添加徽章。

有关应用图标和启动屏幕的更多详情，请参考 Expo 文档。

👉 [创建启动屏幕](https://docs.expo.dev/guides/splash-screens/)
