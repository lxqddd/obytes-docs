启动器附带了超过 10 个 GitHub Actions 工作流，帮助您自动化检查、测试、发布等过程。

所有这些工作流都是自文档化的，并在工作流文件中以注释的形式包含所有详细信息。这种方法使得理解每个工作流的作用及其使用方法变得更容易，无需在文档和工作流文件之间来回切换。

本页列出了我们使用的所有工作流和操作，并对我们用于简化这些工作流的方案进行了更详细的说明。

所有与 GitHub Actions 相关的文件都位于 `.github/workflows` 和 `.github/actions` 文件夹中。

## 操作

您可以将操作视为工作流中用于执行特定任务的函数，它们允许您在多个工作流中重用相同任务而无需复制代码，这使得维护变得更容易。
