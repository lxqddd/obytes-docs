使用Expo，您可以使用`expo-font`库中的`useFont`钩子动态加载字体。使用这种方法，您需要在显示或隐藏启动画面之前等待字体加载完成。

使用Expo 50引入的最新版本的`expo-font`，您可以在`app.config.js`中将`expo-font`作为插件使用，以原生方式加载字体。

要添加自定义字体，您只需要将字体文件放入`assets/fonts`目录，并通过将确切的字体文件路径添加到配置中来更新Expo配置，如下所示：

```js title="app.config.js"
import type { ConfigContext, ExpoConfig } from '@expo/config';

export default ({ config }: ConfigContext): ExpoConfig => ({
  ...config,
  plugins: [
    [
      'expo-font',
      {
        fonts: ['./assets/fonts/Inter.ttf'],
      },
    ],
  ],
});
```

接下来，确保将新字体添加到Tailwind CSS配置中，以便通过`className`使用它

```js title="tailwind.config.js"
const colors = require('./src/components/ui/theme/colors');

/** @type {import('tailwindcss').Config} */
module.exports = {
  // NOTE: Update this to include the paths to all of your component files.
  content: ['./src/**/*.{js,jsx,ts,tsx}'],
  presets: [require('nativewind/preset')],
  theme: {
    extend: {
      fontFamily: {
        inter: ['Inter'],
      },
      colors,
    },
  },
  plugins: [],
};
```

:::info
由于我们以原生方式链接字体，您需要运行`expo prebuild`，然后运行`expo ios`或`expo android`才能使用新字体。
:::

关于使用Tailwind CSS添加字体的更多详情，请参阅[Nativewind文档](https://www.nativewind.dev/v4/tailwind/typography/font-family)。
