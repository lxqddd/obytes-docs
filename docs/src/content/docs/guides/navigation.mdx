[expo-router](https://docs.expo.dev/router/introduction/) 是一个由 Expo 提供的导航库，它简化了在 React Native 应用程序中实现导航的过程。它构建在 React Navigation（一个广泛使用的导航库）之上，并抽象化了管理导航状态和屏幕间转换的大部分复杂性。

Expo Router 中的导航以声明式表达，利用组件来定义应用程序的流程。这种方法使开发者能够直观地构建导航层次结构。

传统的 React Native 项目通常采用这样的结构：单一的根组件通常在 ./App.js 或 ./index.js 中指定。在 Expo Router 的上下文中，通过使用 Root Layout 提供了一种替代方法，该布局位于我们 Demo 中的 `app/_layout.tsx`。因此，我们应用的 `_layout` 部分负责处理整体结构和导航设置。
