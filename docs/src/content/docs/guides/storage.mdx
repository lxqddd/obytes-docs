# 持久化存储

启动器提供了一个简单的存储模块，使用 [react-native-mmkv](https://github.com/mrousavy/react-native-mmkv) 以键值对格式存储数据。我们还添加了一个简单的存储工具来帮助您使用存储模块。

<CodeBlock file="src/lib/storage.tsx" />

`react-native-mmkv` 库提供了各种功能，例如使用 hooks 和为存储的数据添加加密。请随时查看[官方文档](https://github.com/mrousavy/react-native-mmkv)以获取更多信息。
